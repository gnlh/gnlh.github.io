工作区,暂存区,仓库

工作区>>>暂存区

git add 将文件从 工作区 >>> 暂存区

git diff 查看 工作区 和 暂存区 的差异

暂存区>>>仓库

git commit 将文件从 暂存区 >>> 仓库

git diff –cached 查看 暂存区 和仓库 的差异

工作区>>>仓库

git diff HEAD 查看 工作区 和 仓库 的差异

暂存区>>>工作区

git add 的反向命令 git checkout –,撤销工作区的修改,即把暂存区最新版本转移到工作区

仓库>>>暂存区

git commit 的反向命令 git reset HEAD，就是把仓库最新版本转移到暂存区

名词

工作区: Working Directory

版本库: Repository

暂存区(stage)：git add就是把文件添加到暂存区

分支(main)：git commit就是把文件提交到main
